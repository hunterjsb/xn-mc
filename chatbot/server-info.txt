XANDARIS HARDCORE — Server Knowledge

BASICS:
- Server name: Xandaris Hardcore (XHC)
- IP to join: mc.xandaris.space
- Website: xandaris.space
- Discord: discord.gg/UDGfVrTQs6
- Status page: xnmc.statuspage.io
- Minecraft version: 1.21.1 (Paper)
- Max players: 20
- Tagline: "One Life. No Resets. No Mercy."

GAMEPLAY:
- Server type: Hardcore
- Hardcore mode — you die, you're permanently banned. One life. No respawns.
- Vanilla survival — no gameplay-altering mods or plugins. Pure survival experience.
- Anarchy — no rules, no bans (except death), no admins playing favorites. True freedom.
- No resets — the world persists forever. Builds and ruins endure.
- No /home, no /tpa, no economy plugins. You walk or fly.
- Difficulty: Hard
- PvP is on.

WORLD BORDERS:
- Overworld border: 500,000 to -500,000 (1 million blocks wide)
- Nether wall border: 62,500 to -62,500 (125,000 blocks wide)

RANKS (cosmetic only, no gameplay advantages, EULA compliant):
- Supporter ($5/month): colored name in chat
- Champion ($30 one-time): colored name + custom nickname (/nick)
- Mythic ($80 one-time): colored name + /nick + priority queue + custom title + emoticon

VOTING:

DISCORD BOT COMMANDS (in Discord, not in-game):
- /status — check if server is running
- /start, /stop, /restart — server control
- /backup — backup the world
- /mem — system memory usage
- /rcon [command] — send a command to the server

TIPS FOR NEW PLAYERS:
