Revival Bot Benchmarks

Measuring how well AI agents complete Minecraft survival tasks autonomously — from chopping wood to smelting iron.

The Tasks

Each benchmark spawns a revival bot with a specific inventory and goal. The bot must plan and execute a multi-step crafting chain using tool calls — no human guidance.

Wooden Pickaxe
Craft a wooden pickaxe from nothing
Start: empty inventory
mine oak_log craft planks craft crafting_table place table craft sticks craft pickaxe
Cooked Food
Kill an animal and cook the meat
Start: wooden tools, cobblestone, planks, crafting table
craft furnace place furnace kill animal smelt meat
Iron Shears
Mine iron, smelt ingots, craft shears
Start: stone pickaxe, furnace, food, planks
mine iron_ore place furnace smelt raw_iron craft shears

Methodology

  • Bot spawns at world origin, then spreadplayers randomizes to a safe surface location
  • Environment is prepared: oak logs planted (pick), cows summoned (food), iron ore placed (shears)
  • Time set to day, bot teleported to spawn after greeting cycle
  • Goal sent as owner chat message; up to 4 retries if bot goes idle for 20s
  • Success = target item detected in inventory via RCON polling every 5s
  • Timeout: 5 minutes for all benchmarks
  • Metrics: wall-clock time, total tool calls, failed tool calls, LLM chat messages
